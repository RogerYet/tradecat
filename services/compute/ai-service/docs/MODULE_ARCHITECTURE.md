```
    ↓

    ↓
```

## 十、使用示例

### 示例1: 快速AI分析

```python
from ai.src.ai import AICoinQueryManager

# 创建查询管理器
manager = AICoinQueryManager()

# 执行AI分析
result = await manager.query(
    symbol="BTCUSDT",
    interval="15m",
    analysis_type="fast"
)
```
